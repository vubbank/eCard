VÚB eCard plugin

Návod pre implementáciu PHP VÚB eCard pluginu

Predpripravené riešenia

Základné predpoklady

Knižnica je určená pre platformu PHP. Pre svoj úspešný chod, nieje potrebná databáza. Logovanie komunikácie je pomocou textových súborov.

minimálna verzia PHP 5.4

Minimálna verzia, pre ktorú je garantovaná plná funkčnosť je PHP verzie 5.4. Kompatibilita s vyššími verziami je definovaná na základe spätnej kompatibiliti PHP

VÚB eCard prihlasovanie

Nutnou podmienkou funkčnosti, je nastavenie VÚB Client ID (prihlasovanie) a VÚB Store Key (kľúc).

Chybové hlásenia

Systém automaticky vytvára subory, ktoré obsahujú zoznam chýb, ktore sa vyskytnú počas chodu aplikácie.

Tip: API v sebe neuchováva žiadne informácie o platobných kartách klientov

Predpripravené riešenia

Okrem PHP knižnice, pre ktorú je potrebné spraviť integráciu manuálne, balíček obsahuje aj zásuvné moduly pre dvojicu najrozširenejších eshop systémov Magento, Joomla, WooCommerce.

Pohodlné VÚB eCard rozhranie poskytuje aj obľúbený slovenský eshop systém Grandus.

Magento

Magento je e-commerce platforma postavená na open source technológií, ktorá poskytuje online obchodníkom flexibilný systém nákupného procesu, ako aj kontrolu nad vzhľadom, obsahom a funkčnosťou ich internetového obchodu. Magento ponúka výkonný marketing, optimalizáciu pre vyhľadávače a nástroje na správu katalógu, vrátane bezproblémovej integrácie VÚB eCard riešenia.

WooCommerce

WooCommerce je populárny doplnok elektronického obchodu pre WordPress. Umožňuje jednoduché vytváranie a správu obsahu eshopu s primeranou úrovňou flexibility a niekoľkými dôležitými funkciami, ako je správa produktov, bezpečné platby a integrácia dopravy.

Joomla

Joomla! je bezplatný a open-source redakčný systém (CMS) na tvorbu a publikovanie webového obsahu. Je postavený na webovom aplikačnom rámci model-view-controller, ktorý možno použiť nezávisle od CMS. Naše predpripravené riešenie je konkétne pre VirtueMart eshop riešenie do Joomla! systému.

Grandus

Je inovatívna slovenská e-commerce platforma určená pre náročnejších zákaznikov so špecifickými požiadavkami na prepojenie eshopu s vlastným skladovým hospodárstvom, čí inými systémami tretích strán s bohatou ponukou pokročilých eshop funkcionalít. Platforma zároveň ponúka možnosť vývoja vlastných, namieru šitých modulov prevádzkovateľom systému. Plná podpora VÚB eCard riešenia je už základnej výbave.

Vlastná integrácia (PHP)

V prípade, že vlastníte iný systém na platforme PHP, tu sa nachádza popis integrácie.

Vlastná integrácia (JavaScript)

Npm je balíčkovací systém určený pre Javascript. Vďaka nemu je možné jednoducho inštalovať a spravovať balíčky v rámci vášho projektu. NPM baličkovací systém je najpoužívanejší a je súčasťou Vašej inštalácie Node.js.

Tip: Implementáciou podrobnejších informácií o objednávke zvyšujete svoju prehľadnosť o zákazníkoch.

Vlastná integrácia
4 jednoduché kroky

Akákoľvek internetová aplikácia postavená na PHP, móže integrovať novú modernú PHP eCard knižnicu. Stačí postupovať podľa krokov nižšie.

1) Presunutie knižnice do projektu

Je nutné presunúť knižnicu do Vašej aplikácie. Pokial aplikácia podporuje composer, tak knižnica sa načíta automaticky v zmysle integrácie PSR-4.

V prípade, že aplikácia nemá composer, stači načítat autoload.php. Vďaka namespace je možné volať VubEcard objekt.


<?php
require_once('CESTA/KU/KNIZNICI/lib/autoload.php');

/* INICIALIZACIA OBJEKTU, VSTUPNE HODNOTY DODANE VUB */
$vub = new VubEcard\VubEcard(ID, STORE_KEY);

2) Získanie prihlasovacích hodnôt

Pomocou VÚB eCard portálu získame základné identifikačné údaje. Video znázornuje kde sa nachádzajú a ktoré identifikačné údaje sú dôležité. Konkrétne potrebujeme VÚB Client ID a Store Key.

3) Nastavenie základných hodnôt

Získané prihlasovacie údaje, nastavíme v konštruktori VubEcard.php objektu. Okrem Client ID a Store Key je nevyhnutné nastaviť aj návratové url. Návratová url reprezentuje miesto, kam banka presmeruje odpoveď po vykonaní úspešnej/neúspešnej transakcie.



<?php
require_once('CESTA/KU/KNIZNICI/lib/autoload.php');

$vub = new VubEcard\VubEcard(10000001, Kreslo123987, null, true);

/* NAVRATOVA URL DO ESHOPU V PRIPADE OK PLATBY */
$vub->setCallbackUrlSuccesfull('http://mojastranka.domena/spravna-url');

/* NAVRATOVA URL DO ESHOPU V PRIPADE CHYBNEJ PLATBY */
$vub->setCallbackUrlError('http://mojastranka.domena/chybova-url');

4) Informácie o objednávke a sume

Pre správny priebeh platby je nevyhnutné zadať unikátne ID objednávky a celkovú sumu. Suma a ID sú automaticky prenášané do VÚB eCard rozhrania. Funkčnosť je demonštrovaná na videu.



<?php
require_once('CESTA/KU/KNIZNICI/lib/autoload.php');

...

/* NASTAVENIE UDAJOV O OBJEDNAVKE */
$vub->setOrderDetails(123456 /* ID OBJENAVKY */, 1001 /* CELKOVA CENA OBJEDNAVKY */);

/* VYGENERUJE BUTTON NA ODOSLANIE KLIENTA DO BANKY */
echo $vub->generateForm(
  'vubButton',  //NAZOV FORMULARA
